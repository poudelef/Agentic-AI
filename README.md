# Agentic AI 

## Building Blocks of Agentic AI
- Models (LLMs, other AI Models) | Test generation, tool use, web search, text to speech, image analysis
- Tools (APi, Code Exeutation, Information Retrieval) | web search, get real-time data, send email, Databases, Retrieval Augmented Generation (RAG) Basic Calculation

## Agentic Design
1. Reflection (LLM generated output might be checked by another LLMs or Critize or do Unit testing)
2. Tools 
  - Analysis - Code execution, Wolframe Alpha
  - Information Gathering - Web search, Wikipedia, Database access
  - Productivity - Email, Calendar
  - Images - Image generation, OCR
3. Planning  
  
**Prompting is Very important in Agentic Development and better way to learn prompting is you see other people's prompt and learnig from that**
